# SF サブジャンル一覧

ドキュメント調査に基づき抽出された主要な SF サブジャンルの一覧です。

## 1. ハード SF と科学の極北 (Hard SF & Scientific Imagination)

科学的整合性、論理、技術的ディテールを物語の中核に据えるジャンル。

- **宇宙開発・工学 (Engineering)**
  - 具体的な技術ディテールと問題解決を重視。
- **物理・数理 (Physics / Math)**
  - 素粒子物理や天体力学、量子論などの理論的探求を極める。
- **バイオ (Biological)**
  - 遺伝子工学、進化生物学、生態系シミュレーションに基づく。
- **地質・惑星科学 (Planetary / Geological)**
  - テラフォーミングや地殻変動など、惑星規模の物理現象を描く。

## 2. テクノロジーとパンクの美学 (Technology, Punk & Aesthetics)

テクノロジーと社会の関係性、あるいは特定の技術的美学を追求するジャンル群。

- **サイバーパンク (Cyberpunk)**
  - 電脳空間、巨大企業の支配、ハイテク・ローライフ。
- **スチームパンク (Steampunk)**
  - 蒸気機関が極度に発達したレトロフューチャーな世界観。
- **ポスト・サイバーパンク (Post-Cyberpunk)**
  - 技術が日常化した社会。
- **バイオパンク (Biopunk)**
  - 遺伝子工学や生体改造にフォーカスした派生形。
- **ソーラーパンク (Solarpunk)**
  - 環境問題への応答として、自然と技術の共生や持続可能な未来を描く。

## 3. その他の SF サブジャンルと物語形式 (Other Subgenres & Narratives)

宇宙冒険、スペキュレイティブ・フィクション、未知との遭遇など、多様なテーマと構造を持つジャンル群。

- **ロボット工学・AI (Robotics & AI)**
  - ロボット工学三原則、アンドロイド、人工知能の倫理や進化を問う。
- **スペースオペラ (Space Opera)**
  - 宇宙を舞台にした壮大な冒険活劇。ニュー・スペースオペラを含む。
- **ワイドスクリーン・バロック (Widescreen Baroque)**
  - 奔放な想像力、時間・空間の巨大スケール、破天荒なプロットを特徴とするスペースオペラの変種。
- **ミリタリー SF (Military Science Fiction)**
  - 軍事組織、戦術、兵器技術、兵士の心理を描く。
- **海洋 SF (Marine SF)**
  - 海中都市、深海探査、海洋惑星など、海を主たる舞台とする。
- **サイエンス・ファンタジー (Science Fantasy)**
  - 科学的論理よりも幻想性や神話的要素を重視、あるいは魔法と科学が融合した世界観。
- **タイムトラベル / タイムループ (Time Travel / Time Loop)**
  - 時間移動や、特定の期間を繰り返す「ループもの」。
- **SF ミステリー (Sci-Fi Mystery)**
  - SF 的設定や未来技術をトリック・ガジェットとして用いた謎解き・犯罪捜査もの。
- **歴史改変 SF (Alternate History)**
  - 歴史の分岐点を設定し、異なる経過を辿った世界を描く。
- **ディストピア (Dystopia)**
  - 徹底管理された抑圧社会や、反ユートピア的な悲観的未来を描く。
- **ポストアポカリプス / 終末 SF (Post-Apocalyptic)**
  - 文明崩壊後の世界における生存と再生を描く。
- **災害・パニック SF (Disaster / Panic SF)**
  - 巨大災害、パンデミック、異星人の侵略などによる文明崩壊のプロセスを描く。
- **ニュー・ウェーブ (The New Wave)**
  - 1960 年代の変革運動。内宇宙、心理的・社会的問題、実験的文体を重視。
- **ジェンダー SF / フェミニズム SF**
  - 性差やジェンダー役割についての思考実験を行う。
- **スリップストリーム (Slipstream)**
  - 主流文学とジャンル小説の境界にあり、認知的な不協和や非現実感を描く。
- **ニュー・ウィアード (New Weird)**
  - SF、ファンタジー、ホラーの境界を侵犯する「奇妙な」物語。
- **ユーモア / 不条理 SF (Humorous / Absurdist SF)**
  - SF 的設定を風刺やジョーク、ナンセンスとして扱う。
- **ファーストコンタクト (First Contact)**
  - 地球外知的生命体との初めての遭遇と、意思疎通の困難さを描く。
- **言語 SF (Linguistics SF)**
  - 言語学や記号論をテーマの中核に据え、言語による認識の変容を描く。
- **知性化 (Uplift)**
  - 動物を知性化し、人類のパートナーやライバルとして描く。
- **ジュブナイル SF (Juvenile / YA SF)**
  - 少年少女の冒険と成長を描く。往年の冒険ものの系譜。
