# デザイン提案 v2: High-Density Data Archive

ユーザーの要望である「一覧性の高さ」と「ライトテーマ」を両立し、SF 的な機能美を追求したデザイン案です。
没入型のナビゲーションではなく、**一度に全ての選択肢が見渡せる「コックピットの計器盤」や「作戦指令室のスクリーン」のようなレイアウト**を提案します。

## 1. レイアウト構成案： "The Bento Grid System"

Apple や Microsoft の最新 UI でも採用されている「Bento Box（弁当箱）」スタイルを採用します。
画面全体をグリッド（格子）で区切り、大小様々な長方形のカードを隙間なく配置します。

### Top Page Wireframe (Concept)

```text
+-----------------------------------------------------------------------+
|  HEADER: [ SF SUBGENRES ARCHIVE ]   [Search...]   [About]   [Github]  |
+-----------------------------------------------------------------------+
|  +---------------------+  +----------------------------------------+  |
|  | [1] HARD SF         |  | [2] PUNK AESTHETICS                    |  |
|  | Engineering         |  | Cyberpunk                              |  |
|  | Physics / Math      |  | Post-Cyberpunk                         |  |
|  | Bio                 |  +----------------------+-----------------+  |
|  | Planetary           |  | [3] OTHER GENRES     | [4] PICKUP      |  |
|  |                     |  | Space Opera          | Title: Dune     |  |
|  | (Image: Rosetta)    |  | Time Travel          |                 |  |
|  |                     |  | Mystery              | (Cover Image)   |  |
|  |                     |  | Dystopia             |                 |  |
|  +---------------------+  +----------------------+-----------------+  |
|                                                                       |
|  +-----------------------------------------------------------------+  |
|  | [5] GENRE INDEX (Text List for Quick Access)                    |  |
|  | First Contact | Linguistics | Gender SF | New Weird | ...       |  |
|  +-----------------------------------------------------------------+  |
+-----------------------------------------------------------------------+
```

### 特徴

- **区分け**: 大きな枠（Category）の中に、小さなリンク（Subgenre）をリスト表示。
- **視認性**: スクロール量を最小限に抑え、トップページだけでサイトの全貌がわかる。
- **装飾**: 枠線（Border）を強調し、カードの四隅に小さな「＋」マークや装飾的な識別番号（`01`, `A-2`）を配置して、SF 的な精密さを演出。

## 2. デザインテーマ: "Laboratory White"

### カラー体系

- **Background**: `#F8F9FA` (明るいグレーホワイト) - 紙のような白ではなく、少し無機質な白。
- **Card Background**: `#FFFFFF` (純白) - 背景から浮き上がらせる。
- **Text**: `#333333` (ダークグレー) - 真っ黒（#000）よりも目に優しく、洗練された印象。
- **Border/Grid**: `#E2E8F0` (薄い青グレー) - グリッド線を常に表示し、構造を可視化。
- **Accent Color**: `#0055FF` (インターナショナル・オレンジ または テック・ブルー) - 重要なインタラクション要素にのみ使用。

### タイポグラフィ

- **見出し (En)**: `Chakra Petch` or `Orbitron` - 角張った、テクノロジー感のあるフォント。
- **見出し (Jp)**: `Noto Sans JP` (Bold)
- **本文**: `Noto Sans JP` (Regular)
- **装飾文字**: `Roboto Mono` - 識別番号や日付などに使用。

## 3. インタラクション (UX)

### ホバーエフェクト

ユーザーがマウスを乗せた際、単に色が薄くなるだけでなく、情報が「アクティブ」になったことを明確に伝えます。

- **カード全体**: 枠線の色がアクセントカラーに変化し、わずかにシェドウが濃くなる。
- **テキストリンク**: 左側に `>` マークが出現する、あるいは下線がアニメーションで引かれる。

### 画面遷移なしの体験 (理想)

一覧性を高めるため、極力ページ遷移を減らします。

- サブジャンルをクリックすると、**画面右側（または下部）からパネルがスライドイン**して詳細を表示。
- ユーザーは元のリスト（左側）を見ながら、次々と別のジャンルをクリックして詳細閲覧が可能。

## 4. コンポーネント案

### Genre Card (ジャンルカード)

- ヘッダー: 英語タイトル（`HARD SCIENCE FICTION`） + アイコン
- ボディ: サブジャンルのリスト（リンク付き）
- フッター: 短い解説文（例：「科学的整合性を重視する SF の王道」）

### Book List (書籍リスト - 詳細パネル内)

- リスト形式（List View）を採用。
- 左: 表紙サムネイル（小）
- 中央: タイトル・著者名・出版年
- 右: 購入ボタン（アイコンのみのシンプルなもの）

---

この「Bento Grid Layout」と「スライドインパネル」の組み合わせにより、**カタログをパラパラとめくるような一覧性と、データベースを検索するような機能性**を両立させます。
